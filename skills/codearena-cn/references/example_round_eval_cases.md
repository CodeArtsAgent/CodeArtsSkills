### 原始 prompt(本用例之源)

> 报销管理系统。**功能:** 登录/登出;账号管理含 Administrator + Employee 两类角色,管理员管理员工账号,默认管理员 `admin/123456`;报销管理——员工创建/查看/撤销单据,管理员查看/审批/驳回;每张单据含发票标题、发票金额、提交人、银行账号、报销发票图片;须考虑安全。**架构:** 前后端分离。前端 `./frontend` —— Vue3 + Vite + Element Plus + Pinia + Vue Router;特性:表单校验、图片上传预览、列表分页、时间格式化。后端 `./backend` —— Node.js + Express、RESTful API;数据存本地 JSON 于 `./db`(无数据库);图片存本地文件系统于 `./files`;配置 CORS。**测试:** 开发后做 E2E;生成前后端测试脚本;后端 API 测试;用 skill `playwright-cli` 验证前端 UI。**数据清理:** 实现后清理 db 与文件系统中的全部测试数据。**README:** 含前后端启动命令。
